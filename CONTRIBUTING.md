# Contributing

Suggestions and corrections are very welcome.

## Suggesting a tool

Open an [Issue](../../issues/new) with the label **`tool suggestion`** and include:

| Field | Notes |
|---|---|
| **Tool name** | Official name as used in publications |
| **Category** | Closest matching category from the list, or propose a new one |
| **Link** | GitHub / Bitbucket / homepage |
| **One-line description** | What the tool does and why it is relevant to short-read shotgun metagenomics |

There is no strict inclusion threshold — if it is useful to the community, it belongs here.

## Reporting errors

If a link is dead, a description is wrong, or a tool has been superseded, please open an Issue with the label **`correction`**.

## Pull requests

PRs that directly edit `tools.csv` or `README.md` are also welcome. Please keep the CSV sorted alphabetically within each category and regenerate the README table accordingly if you update the CSV manually.

## Scope

The list is primarily aimed at **short-read shotgun metagenomics** (Illumina). Long-read and hybrid tools are included where they are widely used or complement short-read workflows. Amplicon (16S/ITS) tools are generally out of scope unless they serve a clear dual role.
